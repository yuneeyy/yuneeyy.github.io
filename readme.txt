HOW TO ADD YOUR IMAGES
=======================
Just drop image files into this /images folder using the exact filenames
below. The site will pick them up automatically — no HTML/CSS editing
needed. Until a file exists, that spot shows a grey placeholder labelled
with its filename, so you always know what goes where.

Recommended: .jpg or .png, roughly the aspect ratio noted (crops to fill).
